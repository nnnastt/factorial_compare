# Сравнение производительности реализаций факториала

## 📋 Описание проекта

Программа сравнивает скорость работы четырёх различных способов вычисления факториала:

| Реализация | Описание |
|------------|----------|
| `math.factorial` | Встроенная функция Python (написана на C) |
| Итеративная | Вычисление через цикл `for` |
| Рекурсивная | Функция вызывает саму себя |
| Мемоизированная | Рекурсия с сохранением результатов в кэш |

## 🎯 Цель работы

- Изучить различные подходы к решению одной задачи
- Сравнить производительность алгоритмов
- Проанализировать преимущества и недостатки каждого метода
